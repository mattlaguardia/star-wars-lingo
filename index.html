Star Wars Trivia

Welcome to my Star Wars Character Trivia Game!

The game is pretty simple, you will just want to guess what character you think the board is hiding from you. You are timed so try to beat the clock! The empty black box means there is a space in the character's name, and to help you out I have provided you with the first letter and hint button if you need help (I may have also set some triggers to show you some letters if you need some extra help).

Character can be from any of the seven movies! Good Luck!
